Types

LogLevel

 enum class LogLevel

The level at which a message is output. Each LogLevel has a different level of severity, ranging from TRACE to ERROR. In addition, each LogLevel has an assigned ANSI color transform for use with ConsoleWriter.

LogMessage

 sealed class LogMessage

The wrapper class around a log message. Implementations must include an override of the message text, and can include accompanying data.

Logger

 object Logger

The logger singleton. This object is both used for logging and for logger configuration.

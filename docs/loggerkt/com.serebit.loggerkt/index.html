Package com.serebit.loggerkt

Types

FormatterPayload

 data class FormatterPayload

The payload that is sent to Logger.formatter when creating a new LogMessage. Contains all the information needed to create a log message, including the time, the thread name, the class name, the method name, the log level, and the message text.

LeveledLogMessage

 data class LeveledLogMessage : LogMessage

An implementation of LogMessage that includes the LogLevel of the message, along with the message text.

LogLevel

 enum class LogLevel

The level at which a message is output. Each LogLevel has a different level of severity, ranging from LogLevel.TRACE to LogLevel.ERROR. In addition, each LogLevel has an assigned ANSI color transform for use with ConsoleWriter.

LogMessage

 sealed class LogMessage

The wrapper class around a log message. Implementations must include an override of the message text, and can include accompanying data.

Logger

 object Logger

The logger singleton. This object is both used for logging and for logger configuration.

SimpleLogMessage

 data class SimpleLogMessage : LogMessage

The simplest implementation of LogMessage, with its only argument being the message text, and no accompanying data.

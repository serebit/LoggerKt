FormatterPayload(time: OffsetDateTime, threadName: String, className: String, methodName: String, level: LogLevel, message: String)

The payload that is sent to Logger.formatter when creating a new LogMessage. Contains all the information needed to create a log message, including the time, the thread name, the class name, the method name, the log level, and the message text.

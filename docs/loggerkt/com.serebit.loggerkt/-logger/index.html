Logger

object Logger

The logger singleton. This object is both used for logging and for logger configuration.

Properties

async

 var async: Boolean

Determines whether logs should be written asynchronously via coroutines. While this does provide significant performance improvements, logs just before a program exit may not be written, so this defaults to false.

formatter

 var formatter: (FormatterPayload) -> String

The log message formatter.

level

 var level: LogLevel

The LogLevel from which the logger will output log messages. Defaults to LogLevel.INFO.

writer

 var writer: LogWriter

The LogWriter that will be used to output log messages. Can be any predefined LogWriter, or a custom implementation.

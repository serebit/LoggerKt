All Types

com.serebit.loggerkt.writers.ConsoleWriter

An implementation of LogWriter that outputs to the console. This writer's output is color coded using ANSI escape codes by default, unless the operating system is Windows.
com.serebit.loggerkt.writers.FileWriter

An implementation of LogWriter that outputs to a file at the given path. Can be set to either append to the file at the given path, or overwrite it entirely.
com.serebit.loggerkt.formatting.FormatterPayload

The payload that is sent to Logger.formatter when creating a new LogMessage. Contains all the information needed to create a log message, including the time, the thread name, the class name, the method name, the log level, and the message text.
com.serebit.loggerkt.LogLevel

The level at which a message is output. Each LogLevel has a different level of severity, ranging from LogLevel.TRACE to LogLevel.ERROR. In addition, each LogLevel has an assigned ANSI color transform for use with ConsoleWriter.
com.serebit.loggerkt.LogMessage

The wrapper class around a log message. Implementations must include an override of the message text, and can include accompanying data.
com.serebit.loggerkt.writers.LogWriter

Defines an object that is capable of taking LogMessages and outputting them.
com.serebit.loggerkt.Logger

The logger singleton. This object is both used for logging and for logger configuration.
com.serebit.loggerkt.writers.MultiWriter

An implementation of LogWriter that contains multiple LogWriters. This writer passes the log messages through to its internal writers, allowing for multiple log outputs.
com.serebit.loggerkt.writers.PrintStreamWriter

An implementation of LogWriter that outputs to the given stream.

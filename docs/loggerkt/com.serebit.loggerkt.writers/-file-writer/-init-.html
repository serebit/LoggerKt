FileWriter(path: String, overwrite: Boolean = true)

Creates an instance writing to a file at the given path.

Parameters

path - The path at which the file will reside. If the path starts with a slash, the file will be created from the given path itself. Otherwise, the file will be created at a location relative to the parent folder of the classpath.

overwrite - Whether or not the file at the specified path will be overwritten. Defaults to true.
